RECURRING NOTE -- LOCAL DEV CACHING
================================================================================
The local "python -m http.server" sends no cache-control headers, so browsers may
keep serving a cached simulation.js / styles.css after edits. If a change does not
appear, HARD-REFRESH: Ctrl + Shift + R (Windows/Linux) or Cmd + Shift + R (macOS),
or open DevTools -> Network -> "Disable cache". This is a local-dev artifact only;
the production cloud host serves proper cache headers.

All changes above are confined to index.html, styles/styles.css, and simulation.js.
The KL-UNL foundation files remain unchanged except for the one documented JSON
syntax repair to contents.json (see CONVERSION_NOTES.md).
